# Lancement

```
cd ./site
npm run dev
```

> ℹ️ : NPM est requis

# Structure

```
├───📄 index.html : fichier de redirection
├───📄 README.md : ce fichier
└───📁 site
    ├───📁node_modules : modules node.js
    ├──📁public : ressources publiques
    └───📁src
        ├───📁assets
        │    ├───📁css : fichiers css
        │    ├───📁docs : documents
        │    ├───📁fonts : polices d'écritures
        │    ├───📁img : images
        │    └───📁video : vidéos
        ├───📁 components : composants
        ├───📁 router : routeur
        ├───📁 stores : gestionnaires d'états
        │   └───📁 constants : constantes des gestionnaires d'états
        └───📁 views : vues
```
